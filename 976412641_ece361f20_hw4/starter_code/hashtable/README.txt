README.txt - README file for MLS_Helpers directory
--------------------------------------------------
This is based on the code that was provided for ECE 361 Fall 2020 Homework #3.  You
will need to modify the code to use it for Homework #4.   The data file
for Homework #4 contains an additional string at the end of the record that
contains the "nickname" for the team.

To create a key for the Homework #4 hash table your application needs
to concatenate the nickname in the file with either "WEST" (Western conference)
or "EAST" (Eastern conference).  I have provided a function that will convert a string to uppercase
